# 部屋を検索する
・ユースケース: 部屋を検索する
・アクタ: 利用者
・目的: 利用者が希望する日程と利用人数について空いている部屋を検索する。
・事前条件: （なし）
・事後条件: アクタが入力した日程と利用人数を満たす、空いている部屋の一覧が表示されている。
・基本系列
    - 1. アクタは、システムに、部屋を探す旨を示す。
    - 2. システムは、アクタに、日程の入力を促す。
    - 3. アクタは、システムに、希望する日程と利用人数を条件として入力する。
    - 4. システムは、アクタに、入力された日程と利用人数を満たす、空いている部屋の一覧を示す。表示項目は、日時、利用人数、部屋種別、値段、空室数とする。
・代替系列
    - 基本系列4. システムは、該当する部屋がない場合は、警告する。
・備考
    - 部屋種別は、エコノミールームとデラックスルーム、スイートルームの3つである。



# 部屋を予約する
・ユースケース：部屋を予約する
・アクタ：利用者
・目的：利用者が、希望する日程で空いている部屋を予約する
・事前条件：指定された部屋一覧が表示されている
・事後条件：予約が確定している
・基本系列
    - 1. アクタは、システムに、部屋を指定する。
    - 2. システムは、指定された部屋を予約し、予約日時と予約番号、人数を記録する。その際、部屋の空室数を1つ減らす。
    - 3. システムは、アクタに、予約された部屋の料金と予約番号を示す。
・代替系列：(なし)
・備考：(なし)



# 部屋を登録する
・ユースケース：部屋を登録する
・アクタ：ホテル受付係(フロント)
・目的：アクタが、指定された日程、人数、部屋種別の部屋を登録する。
・事前条件：利用者が部屋を予約している。
・事後条件：利用者の予約した日程、人数、部屋種別の部屋が登録されている。
・基本系列
    - 1. アクタは、システムに、部屋を登録する旨を示す。
    - 2. システムは、アクタに、予約番号の入力を促す。
    - 3. アクタは、システムに、予約番号を入力する。
    - 4. システムは、指定された部屋を登録する。
    - 5. システムは、アクタに、登録済みの全ての部屋を示す。表示項目は、チェックイン日、チェックアウト日、人数、部屋種別とする。
・代替系列：(なし)
・備考：(なし)



# チェックインの手続きをする
・ユースケース: チェックインの手続き
・アクタ: ホテル受付係(フロント)
・目的: アクタが、利用者が宿泊する部屋のチェックイン手続きをする
・事前条件: (なし)
・事後条件: 利用者のチェックインが完了している。
・基本系列
    - 1. アクタは、利用者に、予約番号の伝達を促す。
    - 2. 利用者は、アクタに、予約番号を伝達する。
    - 3. アクタは、システムに、チェックイン手続きを行う旨を示す。
    - 4. システムは、アクタに、予約番号の入力を促す。
    - 5. アクタは、システムに、予約番号を入力する。
    - 6. システムは、チェックイン手続きを行い、アクタに部屋番号を伝達する。
    - 7. アクタは、利用者に、部屋番号を伝達する。
・代替系列
	- 基本系列5. システムは、予約番号が存在しない、またはその予約番号においてチェックイン手続きが完了している場合、警告する。
・備考
	- (なし)



# チェックアウトの手続きをする
・ユースケース: チェックアウトの手続き
・アクタ: ホテル受付係(フロント)
・目的: アクタが、利用者が宿泊していた部屋のチェックアウト手続きをする。
・事前条件: (なし)
・事後条件: 利用者のチェックアウト手続きが完了している。
・基本系列
	- 1. アクタは、利用者に、部屋番号の伝達を促す。
	- 2. 利用者は、アクタに、部屋番号を伝達する。
	- 3. アクタは、システムに、チェックアウト手続きをする旨を示す。
	- 4. システムは、アクタに、部屋番号の入力を促す。
	- 5. アクタは、システムに、部屋番号を入力する。
	- 6. システムは、アクタに、入力された部屋の宿泊料を示す。
	- 7. システムは、アクタに、支払いの手続きの完了を促す。
	- 8. アクタは、利用者に、宿泊料の支払いを促す。
	- 9. 利用者は、アクタに、宿泊料を支払う。
	- 10. アクタは、システムに、宿泊料の支払いが完了された旨を示す。
	- 11. システムは、アクタに、チェックアウト手続きが完了したことを示す。
    - 12. アクタは、利用者に、チェックアウト手続きが完了したことを伝える。
・代替系列: (なし)
・備考: (なし)
